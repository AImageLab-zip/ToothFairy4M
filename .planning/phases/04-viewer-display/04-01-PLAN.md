<task type="auto">
  <name>Task 2: Create NiiVueViewer wrapper class</name>
  <files>static/js/modality_viewers/niivue_viewer.js</files>
  <action>
Create a new file `static/js/modality_viewers/niivue_viewer.js` with an ES6 class wrapper around NiiVue.

The class should:
1. Accept a containerId in constructor (the canvas element ID)
2. Store the NiiVue instance and current orientation
3. Provide `init(modalitySlug, fileBlob)` method that:
   - Creates NiiVue instance with multiplanar: false (single-view mode)
   - Loads the volume from the blob
   - Sets default orientation to axial
   - Marks initialized = true
4. Provide `setOrientation(orientation)` method that:
   - Accepts 'axial', 'sagittal', or 'coronal'
   - Calls nv.setSliceType() with appropriate constant
   - Updates currentOrientation property
5. Provide `getSliceIndex()` method for Phase 5 sync
6. Provide `setSliceIndex(index)` method for Phase 5 sync
7. Provide `dispose()` method that cleans up the NiiVue instance

Key implementation notes from research:
- NiiVue slice type constants: nv.sliceTypeAxial, nv.sliceTypeSagittal, nv.sliceTypeCoronal
- Use `await nv.loadVolumes([{url: fileBlob, name: modalitySlug}])` for blob loading
- For mouse wheel scrolling: NiiVue handles this natively (built-in)
- For caching: Keep NiiVue instance alive in caller's state

Expose as `window.NiiVueViewer` for viewer_grid.js to use.
  </action>
  <verify>
Add niivue_viewer.js script to template and test in console:
```javascript

canvas.id = 'test-canvas';
document.body.appendChild(canvas);
const viewer = new window.NiiVueViewer('test-canvas');
console.log(typeof viewer.init === 'function' && typeof viewer.setOrientation === 'function');
// Should output: true
```
  </verify>
  <done>NiiVueViewer class is defined with init(), setOrientation(), and dispose() methods</done>
</task>
