  <files>static/js/viewer_grid.js</files>
  <action>
Modify the `loadModalityInWindow` function to use NiiVueViewer instead of VolumeViewer.

Key changes:

1. Update windowStates structure to track NiiVue instance:
```javascript
windowStates[windowIndex] = {
    modality: null,
    loading: false,
    error: null,
    fileId: null,
    niivueInstance: null,      // Changed from viewerInstance
    currentOrientation: 'axial' // Track orientation for Phase 5
};
```

2. Replace the viewer container HTML with a simple canvas + orientation menu:
```javascript
const canvasId = `niivue-canvas-${windowIndex}`;
const viewerHTML = `
    <canvas id="${canvasId}" class="niivue-canvas"></canvas>
    <div class="orientation-menu">
        <button class="orientation-btn active" data-orientation="axial">A</button>
        <button class="orientation-btn" data-orientation="sagittal">S</button>
        <button class="orientation-btn" data-orientation="coronal">C</button>
    </div>
`;
```

3. Fetch file blob from API BEFORE creating NiiVue (NiiVue needs blob, not modality slug):
```javascript
const response = await fetch(`/api/processing/files/serve/${fileId}/`);
if (!response.ok) throw new Error(`HTTP ${response.status}`);
const fileBlob = await response.blob();
```

4. Create NiiVueViewer instance and init with blob:
```javascript
const viewer = new window.NiiVueViewer(canvasId);
await viewer.init(modality, fileBlob);
```

5. Attach orientation menu event handlers:
```javascript
const menuBtns = windowEl.querySelectorAll('.orientation-btn');
menuBtns.forEach(btn => {
    btn.addEventListener('click', () => {
        const orientation = btn.dataset.orientation;
        viewer.setOrientation(orientation);
        menuBtns.forEach(b => b.classList.remove('active'));
        btn.classList.add('active');
        windowStates[windowIndex].currentOrientation = orientation;
    });
});
```

6. Update state with niivueInstance:
```javascript
windowStates[windowIndex].niivueInstance = viewer;
```

7. Remove the waitForViewerReady polling (NiiVue init is async, so we await it directly).

8. Update clearWindow to dispose niivueInstance:
```javascript
if (state.niivueInstance) {
    state.niivueInstance.dispose();
}
```

Important: Make loadModalityInWindow async (add `async` keyword).
